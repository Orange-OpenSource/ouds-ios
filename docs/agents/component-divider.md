# OUDS iOS – Divider Components

OUDS provides horizontal and vertical dividers, richer than the native iOS `Divider`.
Dividers must use a predefined color token.

**Documentation:**
- Horizontal: <https://ios.unified-design-system.orange.com/documentation/oudscomponents/oudshorizontaldivider>
- Vertical: <https://ios.unified-design-system.orange.com/documentation/oudscomponents/oudsverticaldivider>

## Horizontal divider

```swift
OUDSHorizontalDivider(color: .brandPrimary)
```

## Vertical divider

```swift
OUDSVerticalDivider(color: .brandPrimary)
```
